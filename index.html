An HTML tag is a name, surrounded by a left angle bracket, <, and a right angle bracket, >, that tells your web browser to draw things on the screen and react to events like mouse clicks and button presses.

The special target='_blank' value

Why did we make you use the value _blank in our CoderDojo Ponce Springs link? We did it because when you use _blank, the browser will always create a brand new tab or window whenever the user clicks that link. Try clicking this link a bunch of times. It should always create a new tab or window!

There are only a handful of special targets: _blank, _self, _parent, and _top. Learn all about them on the Webplatform.org attribute guide.

What will happen when a user clicks on this hyperlink two times in a row from the same HTML document: <a href='http://www.coderdojo.com' target='_new'>CoderDojo site!</a>

  • The browser will open a new tab or window after the first click, but nothing will happen after the second click because the special _new target tells the browser to open a new tab or window only one time.
  • The browser will open a new tab or window after each click.
  • The browser will open a new tab or window after the first click, and will warn the user after the second click, informing them that it's rude to click the same link twice in a row so quickly.
  • The browser will open a new tab or window after the first click, and will reload the link in that same newly opened tab or window after the second click.

A CSS selector lets you identify a tag or multiple tags by tag name, by id or class attribute value. You use properties to assign or override various appearance or behavior-related features of the selected tags.

Tag Purpose
table Sets up the start of a table, a container for groups of rows and columns. The table tag can directly contain tr child tags.
tr Creates a table row. Rows stack horizontally on top of each other. The tr tag directly contains child td tags.
td This means "table detail", but you can think of it like a column. You can write multipe td tags next to each other, and they line up side-by-side, or vertically within the row.
th This means "table header", and it has basically the same behavior as the td tag, but it automatically makes any text you put between its starting and closing tags bold and centered. You can change that if you want. You typically place these tags in the very first tr tag so that they identify the meaning of the td tags in the rest of the rows.
